錯題與學習備忘

建議在 GitHub 新增兩個 Markdown 檔：

開啟錯題紀錄　開啟學習備忘

這兩個檔案已預先建立範本。每次做題後只要新增一筆錯題；每次學到新概念或卡關解答後新增一筆備忘。

使用規則

  1. 每天先開這個首頁，選擇一份文件或一個明確任務。
  2. 每次練習後立刻更新 mistakes.md；不要累積到週末才補。
  3. 每週固定一天查看錯題，將重複錯誤列為下週優先主題。
  4. 完成一個里程碑後，手動修改本頁「學習追蹤」的狀態。
